About this Blog

Fun fact, this minimalistic blog is written with Angular with blog posts read from markdown files, hosted on AWS with no Backend. Pros of this approach is that my deployments are extremely simple, cons you don't have the possibility to leave a comment and would have to email me instead (the irony is not lost on me, a java dev who couldn't be bothered to set up a backend).

In this blog I mostly just write down things I find useful and might want to lookup in the future. Occasionally the posts seem to be useful for other people as well, which is always nice.
